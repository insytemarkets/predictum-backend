# Predictum.markets - Polymarket Scanner & Analytics Platform

## Project Overview
Build a standalone, production-ready Polymarket scanner platform focused on:
- Real-time market scanning and opportunity detection
- Advanced analytics and metrics visualization
- Edge detection (spreads, arbitrage, negative risk, volume anomalies)
- Beautiful, modern UI optimized for traders and analysts
- Comprehensive data pipeline with proper rate limiting

## Tech Stack
- **Frontend:** React + TypeScript + Tailwind CSS
- **Backend:** Python (Flask/FastAPI) for data workers
- **Database:** Supabase (PostgreSQL)
- **Real-time:** Supabase Realtime subscriptions
- **Deployment:** Vercel (frontend) + Render (workers)
